Objetivo: Reavaliação periódica do corpo docente do programa a fim de acompanhar a manutenção dos parâmetros exigidos pela CAPES – Área de Medicina II e readequar a composição do grupo. São considerados os docentes permanentes (DP) e docentes colaboradores (DC), com base nos mesmos parâmetros

Indicadores para período corrente

PPQ = Produção ponderada pela estratificação Qualis Periódicos (total de artigos e pontuação requerida para nível 7, considerando o qualis2020)
-> 600 pontos totais (ou 150/ano), com 70% de Docentes Permantentes com 600 ou mais;
-> 50% dos Docentes Permantentes correspondendo aos estratos A1, A2 e A3;
-> 70% dos Docentes Permantentes devem ter publicado 3 ou mais artigos A (pelo menos 2 deles em A1); pelo menos 4 em A2
-> Discentes devem ter produção de 300 pontos/4 anos ou 75/ano
=> Meta: PPQ > 600 pontos para 70% dos discentes do programa no período, atendidos os outros qualificadores

PPD = Percentil de publicações com discentes do programa no período
-> NPCD = Número de publicações com discentes
-> NPSD = Número de publicações sem discentes
-> Cálculo: PCD = NPCD / (NPCD+NPSD) * 100
=> Meta: PPD > 70% de participação discentes na lista de autores

TIT = Pontuação ponderada para orientação de mestres e doutores no período
-> 3 pontos para cada orientação concluídas com discentes titulados (Mestrado e Doutorado)
-> 1 ponto para cada orientações em andamento (Mestrado e Doutorado)
-> Distribuição máxima de orientações em andamento no período 04 discentes por orientador
=> Meta: TIT > 600 pontos para 70% dos discentes do programa no período, atendidos os outros qualificadores

Publicações - Quantidade de Artigos completos

Quantidade de publicações em barras verticais ordenáveis

Essa variação de um gráfico de barras simples adiciona classificação com atraso escalonado e translucidez para melhorar a legibilidade durante a transição. Use a caixa de seleção no canto superior direito para ativar ou desativar a classificação. Durante a classificação, as barras mantêm suas cores padrão para melhor visibilidade. Há um preenchimento lateral adicional adicionado à área do gráfico, a distância entre as barras e seu tamanho são totalmente personalizáveis.

Quantidade de publicações por Papel em barras horizontais ordenáveis

Outro exemplo demonstrando como classificar um gráfico de barras horizontais e animar a reordenação. O atraso escalonado facilita o acompanhamento de compassos individuais. Como no exemplo do gráfico de barras horizontais, as barras mantêm suas cores originais. O texto de valor opcional pode ser colocado nos lados esquerdo e direito ou centralizado. A altura da barra depende da altura do contêiner do gráfico e do número da barra exibido.

Detalhes estratos por docente

Esta variação de um gráfico de barras empilhadas adiciona uma opção para alterar o gráfico de barras empilhadas para múltiplo em tempo real, selecionando o botão de opção. Os rótulos do gráfico se movem de acordo com a posição do grupo de barras. Este tipo de layout e transições estão disponíveis para ambas as direções da barra - vertical e horizontal. Ele usa um conjunto de dados para ambos os tipos de gráfico. Os rótulos dos eixos verticais são opcionais e ocultos por padrão.

Histograma das quantidades de publicações por docente

Este exemplo mostra o histograma básico com dica de ferramenta opcional. Os dados são gerados aleatoriamente. Os valores são agrupados em intervalos regulares usando o layout de histograma do D3. O eixo x usa uma escala linear, de forma que os valores de marcação apareçam entre as barras; isso fornece uma melhor indicação de que cada barra representa a contagem de valores entre os valores de marcação ao redor.
