Avaliação contínua para avaliar e readequar a composição do corpo docente do programa conforme parâmetros exigidos pela CAPES – Área de Medicina II.
São considerados os mesmos parâmetros como base para avaliar docentes permanentes (DP) e docentes colaboradores (DC).

Indicadores para período corrente

PPQ = Produção Docente ponderada por Qualis Periódicos (requerida para nível 7, considerando o qualis2020)
-> Objetivo: Nível 7, significa ter 70% de seus docentes dentro dos 40% mais produtivos em Medicina II;
 => Programa: 70% dos Docentes Permantentes devem atingir a meta de pontuação;
=> Docente: PPQ > 1000 pontos de impacto no quadriênio e 04 produções A1;
=> Discente: PPQ > 500 pontos no quadriênio e 125/ano;
PPD = Percentil de publicações com discentes do programa no período
=> Meta: PPD > 70% dos artigos do programa com participação discente na lista de autores;
-> NPCD = Número de publicações com discentes
-> NPSD = Número de publicações sem discentes
-> Cálculo: PCD = NPCD / (NPCD+NPSD) * 100
TIT = Pontuação ponderada para orientação de mestres e doutores no período
=> Meta: TIT > 4 anos em média para doutorado;
-> 3 pontos para cada orientação concluídas com discentes titulados (Mestrado e Doutorado)
-> 1 ponto para cada orientações em andamento (Mestrado e Doutorado)
-> Distribuição máxima de orientações em andamento no período 04 discentes por orientador
